Security Services for the Library Report PPL18-012

Moved by Garth Wedlock

That the Library Board approve Report PPL18-012, dated February 13, 2018 of the Library CEO, as follows:

a) That Security Services be contracted for the Library for the Aylmer Street

location as a pilot project at an estimated cost of $27,000; and,

b) That the Board evaluate the pilot project at the September Board meeting.

The Library CEO advised the Library Board that various groups/individuals have expressed an interest in renting space in the foyer for displaying materials and information. The Library Board discussed the issue of renting table space in the library entrance.
